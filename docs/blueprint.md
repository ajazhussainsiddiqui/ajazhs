## Core Features:

- Markdown Rendering: Render markdown content from Firestore, supporting HTML, tables, and GitHub Flavored Markdown.
- Block-Based Content: Enable content creation using a flexible block system (text, image, spacer).
- Responsive Layouts: Provide various responsive layouts (single, two, three-column, grid) for different content arrangements.
- Style Token System: Implement a style token system for consistent styling (fontSize, fontWeight, spacing).
- Firestore Data Integration: Initialize Firestore and manage data services using the provided API keys. Safe error handling will be provided.

## Style Guidelines:

- Primary color: Dark slate blue (#374151) to convey sophistication and trust (support dark mode).
- Background color: Very light gray (#F9FAFB), nearly white, for a clean and spacious feel.
- Accent color: Muted teal (#4ADE80) for highlights and calls to action, adding a touch of modern freshness.
- Body font: 'Inter', sans-serif, for a modern, machined look; headlines: 'Space Grotesk', sans-serif, for computerized feel.
- Simple, geometric icons that align with the site's modern aesthetic.
- Clean, grid-based layouts that emphasize content and readability.
- Subtle transitions and animations to enhance user experience without being distracting.
